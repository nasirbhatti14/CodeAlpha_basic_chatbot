# Basic ChatBot

A simple rule-based chatbot built using Python. This chatbot responds to predefined user inputs and demonstrates the use of basic Python concepts such as functions, loops, conditional statements, and input/output operations.

## Features

- Responds to basic greetings and questions.
- Uses predefined responses.
- Runs continuously until the user types `bye`.
- Beginner-friendly Python project.

## Concepts Used

- Functions
- While Loops
- If-Elif-Else Statements
- User Input
- Output Printing

## Supported Commands

| User Input | ChatBot Response |
|------------|------------------|
| hello | Hi! |
| how are you | I'm fine, thanks for asking! |
| what is your name | My name is ChatBot. |
| bye | Goodbye! |

## How to Run

1. Make sure Python is installed on your system.
2. Clone this repository:

```bash
git clone https://github.com/your-username/basic-chatbot.git
```

## Navigate to the project directory:
```
cd basic-chatbot
```
## Run the Python file:
```
python chatbot.py
```
## Example
```
========= Basic ChatBot =========

ChatBot: Hello! Type 'bye' to exit.

You: hello
ChatBot: Hi!

You: how are you
ChatBot: I'm fine, thanks for asking!

You: bye
ChatBot: Goodbye!
```
